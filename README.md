# Medicine E-Commerce Platform

A full-featured **medicine store platform** built with **Next.js, TypeScript, BetterAuth, and Zod3**. This project supports multiple user roles — **Customer, Seller, Admin** — with role-specific functionality and a smooth, secure experience.


### General
- **Authentication & Authorization** using BetterAuth.
- **Form validation** with **Zod3**.
- **Type-safe API routes** with Next.js and TypeScript.
- **Role-based access control** for Customer, Seller, and Admin.
- **Responsive design** for web and mobile devices.

### Customer
- Browse & search medicines.
- Add items to cart.
- Place orders and track order status.
- Add reviews and ratings for purchased products.
- View order history.

### Seller
- Add new medicines with details (name, price, description, image, stock).
- Edit existing products.
- Delete products.
- Manage order status (deliver, cancel).
- View product and sales analytics.

### Admin
- Manage all users: ban/unban.
- View all orders and update status.
- Manage total inventory (all medicines in store).
- Analytics dashboard for total orders, sales, and user activity.
